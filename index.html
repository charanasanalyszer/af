⬆ Import Fee Payments from Excel

Upload an Excel file with fee payment records. Existing records for the same student/term/year will be updated with new payments.

Click or drag Excel/CSV file

Columns: AdmNo, Term, Year, TotalFee, AmountPaid, PaymentDate, PaymentMode, ReceiptNo, Notes

Import Instructions

Required columns:

  • AdmNo — Student admission number (must exist in system)
  • Term — e.g. Term 1, Term 2, Term 3
  • Year — e.g. 2025
  • TotalFee — Total fee amount for the term (KES)
  • AmountPaid — Amount being paid in this record (KES)

Optional columns: PaymentDate, PaymentMode (Cash/Mpesa/Bank/Cheque), ReceiptNo, Notes

Students not found in the system will be skipped. Download the template to see the exact format.
